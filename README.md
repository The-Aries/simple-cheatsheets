# simple-cheatsheets

`simple-cheatsheets` is a pure static website project centered around cheatsheets. It is primarily intended for personal use, serving the maintainer's own learning, organization, and reference needs, while also being accessible to web users.

This repository is used for:

1. Storing the project source code.
2. Recording the current project status and next steps.

## Usage

Designed for users who already have some basic knowledge and need a quick way to look up commands by topic.

---

## Current Technical Direction

HTML + CSS + JavaScript + JSON + Excel

The current implementation approach is as follows:

- Page structure is built with HTML.
- Page styling is handled with CSS, with a priority on keeping the site minimal and purely static.
- Page logic and data loading are handled with vanilla JavaScript.
- JSON is used as the runtime data format.
- Cheatsheet content is primarily entered through Excel, then organized or converted into JSON for page usage.
