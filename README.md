# simple-cheatsheets

`simple-cheatsheets` 是一个以 cheatsheet 为核心、以个人使用为主的纯静态站点项目，服务于维护者个人的学习、整理与查阅，也开放给网络用户使用。

本仓库同时用于：

1. 存放项目源码。
2. 记录项目状态与下一步。

---

## 计划覆盖的主题

- Markdown
- Git
- Linux
- MATLAB 

---

## 使用方式

适合已具备基础、需要快速查阅的人，快速按主题查找命令

---

## 当前技术方向

HTML + CSS + JavaScript + JSON + Excel

当前实现思路如下：

- 页面结构使用 HTML。
- 页面样式使用 CSS，优先保持极简、纯静态。
- 页面逻辑与数据读取使用原生 JavaScript。
- 页面运行时使用 JSON 作为数据格式。
- cheatsheet 内容优先通过 Excel 录入，再整理或转换为 JSON 供页面使用。
